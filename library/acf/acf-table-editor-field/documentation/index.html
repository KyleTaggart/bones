A) Description - top

This plugin adds a new ACF field that creates an Excel like grid table that can be populated by admin users. The data can then be rendered as an HTML table on the frontend of your site.

This plugin requires the free plugin Advanced Custom Fields

B) Requirements - top

Before we go ahead and install the plugin you need to make sure your WordPress install will be able to handle the plugin. You need to have at least the following running on your server:

  • WordPress v3.0+
  • Advanced Custom Fields v4.0+

If you don’t have these or you don’t know what these are, it’s worth getting in touch with your hosting provider just to check.

C) Installation - top

Once you have purchased your copy of the ACF Table Editor Field and downloaded it to your computer you will need to follow these steps to install the plugin in WordPress:

  1. Log in to your WordPress admin backend.
  2. Go to the Plugins > Add New page.
  3. Click on the Upload tab.
  4. On this page click the Choose File button choose the ACF Table Editor Field zip file that you previously downloaded.
  5. Click the Install Now button and the plugin will be installed.
  6. You should now see a ACF Table Editor Field plugin in your list of plugins. Click Activate. 
  7. You should now see a Table Editor field in the ACF field group area.

D) Configuration - top

Creating a table editor field

  1. Navigate to the Custom Fields menu in your dashboard
  2. Click Custom Fields 
  3. Edit an existing Field Group or add a new one
  4. Add a new Field in the group
  5. You should now see a new Field Type under 'Content' of Table Editor 
  6. Fill out the usual field details like label, instructions and required
  7. Set the number of columns the table will have
  8. Set the column headers.Simple - headers are A,B,C,D etc. Custom - you can set the header names below.None - is for now header
  9. If you have selected custom headers, enter the names for each column in the text area. A column name on a new line.
  10. Set how many empty rows there will be at the bottom of the table. (These empty rows will not be rendered but showing one empty row can be helpful to users)
  11. Set if the row numbers will be shown on the table. (These will not be rendered).
  12. Set if the table will fill the width of the metabox
  13. Set if users can use drag down functionality on cells for easy copying
  14. Update the field group

Populating the table

  1. Edit your post, page or custom post type that you have set the ACF field group, with the Table Editor field in, to appear on
  2. You will see the table appear like an Excel sheet. Start populating data
  3. You can add new rows, remove rows and undo and redo text changes
  4. Once finished publish/update the post, page or custom post type
Displaying the table

  1. To render your table in your template use get_field(). The table is rendered as a normal HTML table element and the styling of the table is controlled by your theme not the plugin.
