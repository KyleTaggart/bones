A) Description - top

This plugin adds a new ACF field that automatically generates a screenshot of a website from a link provided from an ACF text field and saves it to your WordPress media library.

This plugin requires the free plugin Advanced Custom Fields

B) Requirements - top

Before we go ahead and install the plugin you need to make sure your WordPress install will be able to handle the plugin. You need to have at least the following running on your server:

  • WordPress v3.0+
  • Advanced Custom Fields v4.0+

If you don’t have these or you don’t know what these are, it’s worth getting in touch with your hosting provider just to check.

C) Installation - top

Once you have purchased your copy of the ACF Website Screenshot and downloaded it to your computer you will need to follow these steps to install the plugin in WordPress:

  1. Log in to your WordPress admin backend.
  2. Go to the Plugins > Add New page.
  3. Click on the Upload tab.
  4. On this page click the Choose File button choose the ACF Website Screenshot zip file that you previously downloaded.
  5. Click the Install Now button and the plugin will be installed.
  6. You should now see a ACF Website Screenshot plugin in your list of plugins. Click Activate. 
  7. You should now see a Website Screenshot field in the ACF field group area.

D) Configuration - top

Creating a website screenshot field

  1. Navigate to the Custom Fields menu in your dashboard
  2. Click Custom Fields 
  3. Edit an existing Field Group or add a new one
  4. Make sure you have a text field created that will store the website URL you want to snapshot. Update the field group
  5. Add a new Field in the group
  6. You should now see a new Field Type under 'Content' of Website Screenshot 
  7. Fill out the usual field details like label, instructions and required
  8. Select the ACF field that will contain the website URL in the Source Website URL Field field
  9. Enter the Width and Height values that the screenshot will be generated at
  10. Set the Return Value and Preview Size as you would a normal ACF image field
  11. Update the field group

Generating a screenshot

  1. Edit your post, page or custom post type that you have set the ACF field group, with the Website Screenshot field in, to appear on
  2. You will see the Screenshot field with a Generate button. Click that to create your website screenshot
  3. The preview image will appear at the size you have selected in the field admin. The image has now been saved to your WordPress media library
  4. You can remove the image or regenerate the screenshot at any time
Displaying the screenshot

  1. Display your screenshot in the same way you would a normal ACF image field
